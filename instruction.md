# Family Barbershop Website Instructions

## Project Goal
Build a modern, professional, and mobile-friendly website for a local business called **Family Barbershop**.

The website should help customers:
- See services and prices
- View barber shop photos
- Read customer reviews
- Contact the shop easily
- Book appointments
- Find the shop location

The design must feel:
- Clean
- Professional
- Trustworthy
- Family-friendly
- Modern barber shop aesthetic

---

# Technology Stack

Use the following technologies:

- Next.js
- React
- Tailwind CSS

Requirements:

- Fully responsive design (mobile-first)
- Clean and maintainable code
- Reusable components
- SEO-friendly structure
- Fast loading performance

---

# Pages Required

Create the following pages:

1. Home
2. About
3. Services
4. Gallery
5. Reviews
6. Contact / Booking

---

# Home Page Sections

The homepage should contain these sections:

### Hero Section
- Strong headline
- Short introduction
- Call-to-action buttons (Book Now / Call Now)

### Why Choose Us
Show benefits like:
- Experienced barbers
- Clean shop
- Friendly atmosphere
- Great service for families

### Featured Services
Display key services with simple cards.

### Reviews Preview
Show customer testimonials.

### Gallery Preview
Display a few photos from the shop.

### Hours & Location
Show business hours and map placeholder.

### Final CTA
Encourage users to call or book.

---

# Services Page

Include common barber services:

- Haircut
- Skin Fade
- Beard Trim
- Kids Haircut
- Line Up
- Hot Towel Shave

Services should be displayed using a **clean card layout**.

Prices should be easy to edit later.

---

# Gallery Page

Create a modern gallery with:

- Responsive image grid
- Hover effects
- Clean layout

Images should be easy to replace later.

---

# Reviews Page

Display testimonials using:

- Card layout
- Star rating visuals
- Clean typography

Add a section encouraging customers to leave reviews.

---

# Contact / Booking Page

Include:

- Phone number
- Address
- Business hours
- Google Maps placeholder
- Contact form
- Booking button
- Social media links

Make it easy for mobile users to contact the shop.

---

# Design Guidelines

The design should be:

- Modern
- Clean
- Premium but welcoming
- Professional barber shop style

Use:

- Strong typography
- Good spacing
- Subtle animations
- Clean color palette

---

# Code Rules

Always follow these rules:

- Use reusable components
- Keep files organized
- Use semantic HTML
- Write clean code
- Add comments where business information should be edited later
- Avoid unnecessary complexity
- Optimize for local business SEO
