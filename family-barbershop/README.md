## Family Barbershop Website

Professional, modern, mobile-friendly website for **Family Barbershop** built with **Next.js (App Router)** and **Tailwind CSS**.

## Getting started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Where to edit content

- **Business info, prices, reviews, gallery**: `src/content/business.ts`
- **Pages**: `src/app/*/page.tsx`
- **Shared layout**: `src/app/layout.tsx`
- **Header/Footer**: `src/components/site/`

## Notes

- **Gallery images** live in `public/gallery/` (currently SVG placeholders).
- **Contact form** is a demo placeholder until you connect it to a form provider or backend.

## Tech

- Next.js + React
- Tailwind CSS
- TypeScript
